# Carewallet × Clinic Chain — Business Case Model

Interactive financial model for the Carewallet clinic chain partnership.

## Deploy to Vercel

1. Push this folder to a GitHub repo
2. Go to [vercel.com/new](https://vercel.com/new)
3. Import the repo → Vercel auto-detects Next.js → Deploy

Or use the CLI:

```bash
npm i -g vercel
vercel
```

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
